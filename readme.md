# vue-http-fast

> simplly http request in vue component

## How to install

```sh
yarn add vue-http-fast
# or
npm i vue-http-fast
```

```js
import vueHttpFast from 'vue-http-fast';

// in Vue 2
Vue.use(vueHttpFast);

// in Vue 3
App.use(vueHttpFast);
```

## How to use

```vue
<http-fast url="">
    ... some html
</http-fast>
```

## Docs

### Params

| Name   | Type   | Description          |
| ------ | ------ | -------------------- |
| url    | String | url for http request |
| method | String | sss                  |
